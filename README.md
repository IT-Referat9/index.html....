# CyberDash

A high-tech, Cyberpunk-themed IT monitoring dashboard scripted in PowerShell using WPF.

## Quick Start

1. Open PowerShell as Administrator.
2. Run:
   ```powershell
   powershell.exe -ExecutionPolicy Bypass -File .\CyberDashboard.ps1
   ```

## Features

- Real-time CPU & RAM tracking.
- Disk usage monitor.
- Active system service list.
- Premium neon aesthetics.
